# Landing Page Screenshots

Поместите скриншоты в эту папку с следующими именами:

## Необходимые файлы:

1. **compounders-roic.png** - Скриншот Compounders (ROIC) layout
   - Показывает колонки: ROIC % (Latest), ROIC 10Y Avg %, ROIC Volatility %, ROIC Stability Score, ROIC History (10Y)

2. **cashflow-leverage.png** - Скриншот Cashflow & Leverage layout
   - Показывает колонки: Free Cash Flow, FCF Margin %, FCF Margin 10Y Median %, FCF Margin History (10Y), Debt-to-Equity, Interest Coverage, Cash Flow to Debt
   - Используется для разделов "FCF Margins" и "Balance Sheet"

3. **dupont-roe.png** - Скриншот DuPont ROE Decomposition layout
   - Показывает колонки: Net Profit Margin, Asset Turnover, Financial Leverage, ROE %

4. **return-on-risk.png** - Скриншот Return on Risk layout
   - Показывает колонки: 3Y Return/Risk, 5Y Return/Risk, 10Y Return/Risk

5. **dcf-valuation.png** - Скриншот DCF Valuation layout
   - Показывает колонки: Market Cap, Price, Revenue, Rev G 10Y, DCF Enterprise Value, Margin of Safety

6. **reverse-dcf.png** - Скриншот Reverse DCF layout
   - Показывает колонки: Revenue History (10Y), Rev G 10Y, DCF Implied Growth, Model Verdict
   - Используется для разделов "Reverse DCF" и "Long-term Growth"

## Формат файлов:
- Формат: PNG
- Рекомендуется: Высокое качество, но оптимизированные для веба
